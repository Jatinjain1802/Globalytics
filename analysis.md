# Website Redesign Analysis: Globalytics Digital

This document outlines the current state of the [Globalytics Digital](https://globalyticsdigital.in/) website and provides a strategic roadmap for its complete redesign.

---

## 1. Current Website Audit

### **Main Sections**
- **Hero Section**: Focuses on business growth with standard CTAs.
- **Services**: Lists SEO, SMM, Google Ads, and Web Dev.
- **About/Stats**: Displays experience and project counts.
- **Testimonials**: Client feedback (Agency focused).
- **Contact**: A standard form for service queries.

### **Identified Gaps**
- **Missing Institutional Focus**: Despite being an academy, there is zero mention of courses, students, or training programs on the homepage.
- **Generic Aesthetics**: The current design uses standard WordPress/Elementor blocks which lack a unique "Premium" feel.
- **Navigation**: No clear path for prospective students (Academy/Courses).
- **User Journey**: The site targets business owners but completely ignores the student persona.

---

## 2. Redesign Strategy

### **The Dual-Persona Approach**
The new website must cater to two distinct audiences simultaneously:
1.  **Business Clients**: Looking for high-ROI digital marketing services.
2.  **Students/Learners**: Looking to build a career in digital marketing.

### **Proposed Content Structure**
- **Unified Hero**: A "Split" or "Dynamic" hero section that acknowledges both Agency and Academy roles.
- **Service & Course Grids**: Interactive cards with distinct visual cues for services vs. courses.
- **Expertise Section**: Highlighting the team as both practitioners (for clients) and mentors (for students).
- **Success Stories**: Alternating between Client Case Studies and Student Placement success.

---

## 3. Visual & Technical Direction

### **Design Language (Aesthetics)**
- **Theme**: Premium Dark Mode with sleek glassmorphism.
- **Accent Colors**: Electric Cyan (#00C2E8) and Deep Indigo for a "Tech-Forward" feel.
- **Typography**: Outfit or Inter for a modern, geometric look.
- **Micro-Animations**: Framer Motion for smooth entry and hover transitions.

### **Technical Stack**
- **Next.js 14**: Framework (App Router).
- **React**: Library for building components.
- **CSS Modules**: Vanilla CSS with scoping.
- **Lucide React**: Icons.

---

## 4. Educational Roadmap (Learning Focus)

As we build this project, we will focus on:
1.  **React Fundamentals**: Creating reusable UI components (Buttons, Cards, Modals).
2.  **Next.js App Router**: Handling dynamic routing for /services and /courses.
3.  **Modern CSS**: Implementing responsive layouts using Grid and Flexbox.
4.  **State Management**: Using React Hooks (useState, useEffect) for interactive forms.
